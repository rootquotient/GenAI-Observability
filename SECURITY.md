## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the GenAI Observability SDK, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please report security vulnerabilities by emailing [INSERT EMAIL ADDRESS].

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

### Review and Disclosure

We will investigate the report and, if a vulnerability is confirmed, we will work on a patch. We will notify you when the patch is available and credit you in the release notes (unless you prefer to remain anonymous).
